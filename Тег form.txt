 -Тег form требуется для заполнения символами пользователю (тоесть пользователь может взаимдействовать с данным блоком)
 -В этом теге есть внутренние теги, которые необходимо использовать. Например, тег input, самый распространенный. В нем автоматически вводится класс type, в котором мы можем ввести параметр.
Например, type="text" - в данном случае пользователь вводит свой пароль, емейл, логин и т.д., что угодно.
Но в классе type так же есть параметр tel. Выглядит это таким образом: type="tel". В данном случае пользователь имеет право вводит только цифры(номер телефона, например) (!Данная функция действует только в том случае, если пользователь вводит данные со смартфона!)
Поле password. type="password" - проверяет введенный вами пароль (текст будет вводиться, но не будет нам показываться)
Поле email. type="email" - в данном поле необходимо ввести свой емеил (проверяется наличие знака "@" и адреса после нее, например : @gmail.com, @mail.com)

__К полю можно добавить какой-нибудь праметр__ 
Например : обазначить обязательно ли данное поле заполнять или нет. Для этого после type="" необходимо ввести атрибут required. Данный атрибут проверет заполнено ли это поле. Если оно не заполнено, то атрибут не дает отправить форму. Выглядит таким образом : <input type="text, tel и т.д." required>
Так же есть параметр minlength. Данный параметр проверяет минимальное значение символов. Например : <input type="password" minlength="8">. В данном случае пользователю необходимо ввести пароль, содержащий не менее 8 символов.
И так же можно поставить ограничение символов параметром maxlength.
Параметр placeholder. Данный параметр не работает в браузере internet explorer ниже 8-ой версии, хотя сам параметр является важным. Данный параметр - это подпись поля ввода (input). Тоесть пользователю можно указать, что ему нужно ввести в поле. Например,  <input type="text" placeholder="Введите номер телефона">
Textarea - тег, необходимый для введения большого кол-ва текста (используется вместо тега input)
